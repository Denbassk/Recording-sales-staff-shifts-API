# 04. API Эндпоинты

Все эндпоинты требуют JWT-токен в cookie `token`.

## Аутентификация

| Метод | Путь | Описание |
|-------|------|----------|
| POST | `/login` | Вход, устанавливает cookie `token` |
| POST | `/logout` | Выход, удаляет cookie |
| GET | `/me-with-store` | Текущий пользователь + его магазин |

## Смены

| Метод | Путь | Описание |
|-------|------|----------|
| GET | `/shifts` | Список смен (с фильтрами) |
| POST | `/shifts` | Создать смену |
| PUT | `/shifts/:id` | Обновить смену |

## Возвраты (продавец)

| Метод | Путь | Описание |
|-------|------|----------|
| GET | `/lookup?barcode=&store_address=` | Поиск товара по ШК в BigQuery |
| GET | `/search-product?q=&store_address=` | Поиск товара по названию |
| POST | `/returns` | Создать возврат (шапка + позиции) |

## Возвраты (администратор)

| Метод | Путь | Описание |
|-------|------|----------|
| GET | `/returns` | Список возвратов с фильтрами + вложенные позиции |
| GET | `/returns/summary?group_by=supplier\|sku\|store` | Агрегированная сводка |
| POST | `/returns/:id/archive` | Архивировать возврат |
| GET | `/stores` | Список магазинов (для фильтров) |

## Зарплаты

| Метод | Путь | Описание |
|-------|------|----------|
| GET | `/payroll` | Расчёт зарплат |
| POST | `/payroll/export` | Экспорт в Excel |

## Расположение кода

Все эндпоинты системы возвратов находятся в файле **`routes/lookup.js`**, который подключается в `server.cjs` как:
```js
const lookupRouter = require('./routes/lookup');
app.use('/', lookupRouter);
```

Основные смены/зарплаты — в `server.cjs`.
