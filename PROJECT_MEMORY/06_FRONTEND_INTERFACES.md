# 06. Фронтенд-интерфейсы

## Все страницы

| URL | Файл | Аудитория | Назначение |
|-----|------|-----------|------------|
| `/` | `public/index.html` | Все | Главная — смены продавцов |
| `/returns.html` | `public/returns.html` | Продавцы | Оформление возвратов (сканирование) |
| `/admin-returns.html` | `public/admin-returns.html` | Админ/Бухгалтер/Куратор | Просмотр и управление возвратами |
| `/payroll.html` | `public/payroll.html` | Бухгалтер/Админ | Расчёт и экспорт зарплат |
| `/test-auth.html` | `public/test-auth.html` | Разработчик | Тестирование аутентификации |

## Интерфейс продавца — Возвраты (`/returns.html`)

```
┌──────────────────────────────────────────────────────┐
│ 👤 Имя сотрудника • [Адрес магазина]  [0] [❓] [Вихід] │
├──────────────────────────────────────────────────────┤
│                                                        │
│  СКАНУЙТЕ ШТРИХ-КОД                                  │
│  ┌────────────────────────────────────────────────┐  │
│  │         (поле ввода, всегда в фокусе)          │  │
│  └────────────────────────────────────────────────┘  │
│  [плашка статуса последнего сканирования]             │
│                                                        │
│  🔍 ПОШУК ЗА НАЗВОЮ                                  │
│  ┌────────────────────────────────────────────────┐  │
│  │              (поиск по тексту)                  │  │
│  └────────────────────────────────────────────────┘  │
│  [результаты поиска inline]                           │
│                                                        │
│  🛒 Кошик повернення (N)                              │
│  ┌─ название ──────────── кол-во ──── сумма ── [✕] ─┐ │
│  │ ...                                               │ │
│  └───────────────────────────────────────────────── ┘ │
│  Позицій: N   Сума: 0.00 ₴                           │
│           [Очистити]  [Відправити повернення]         │
└──────────────────────────────────────────────────────┘
```

**Модальные окна:**
- `#qtyModal` — кол-во для зелёного товара (кнопки +/−)
- `#confirmModal` — подтверждение для оранж/синего
- `#manualModal` — ручной ввод (жёлтый)
- `#helpModal` — справка по работе

## Интерфейс администратора — Возвраты (`/admin-returns.html`)

```
┌──────────────────────────────────────────────────────┐
│ 🛡️ Имя (admin)                    [🔄] [●] [Вихід]  │
├──────────────────────────────────────────────────────┤
│ [Период с] [по] [Магазин▼] [Статус▼] [Цвет▼] [Застос.]│
│                                           [📥 Експорт]│
├──────────────────────────────────────────────────────┤
│ [Список повернень] [Зведення: постачальник] ...       │
├──────────────────────────────────────────────────────┤
│ Повернень: N  Позицій: N  Сума: 0.00 ₴              │
│                                                        │
│ ┌─ RET-260101-001 ──── Магазин ── Дата ── N поз ──┐  │
│ │  [В архів]                                        │  │
│ │  ▼ (разворачивается — список позиций)             │  │
│ └───────────────────────────────────────────────── ┘  │
└──────────────────────────────────────────────────────┘
```

**Авто-обновление:** каждые 30 секунд (индикатор ● мигает)

## Запуск на кассе (Windows)

`Returns.bat` → открывает Chrome/Edge с URL:
```
https://shifts-api.fly.dev/returns.html?device=DEVICE_KEY
```

`device.json` в корне папки хранит `device_key` конкретной кассы.
